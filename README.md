🌍 EcoSync: Real-time Carbon Footprint Analytics & Optimization
EcoSync is an AI-powered platform that analyzes computational resource usage to predict carbon emissions and recommend optimal scheduling for reduced environmental impact.

Python Streamlit License

Features
📊 Real-time Dashboard: Monitor CPU, RAM usage, and their carbon footprint in real-time
📈 Predictive Analytics: Time-series forecasting of carbon emissions using Prophet
🌿 Green Scheduler: AI-powered task scheduling during low-carbon periods
🌳 Impact Visualization: "Trees saved" equivalents for intuitive understanding
Tech Stack
Component	Technology
Frontend	Streamlit
Data Processing	Pandas, NumPy
Forecasting	Prophet (Facebook)
Visualization	Plotly
System Monitoring	psutil
